## Numerology References

This repository references common numerology systems used for
name-based and date-based number analysis.

The following external resources are provided for educational
and exploratory purposes:

- [Lo Shu Grid Numerology Calculator](https://www.astrohoraz.com/numerology/lo-shu-grid-numerology-calculator)
- [Name Numerology Calculator](https://www.astrohoraz.com/numerology/name-numerology-calculator)
- [Birthday Compatibility](https://www.astrohoraz.com/numerology/birthDay-compatibility)

These tools demonstrate how numbers derived from names and
dates are interpreted using traditional numerology methods.
